## Инструкция по установке:

  * Склонировать репозиторий 

  * Перейти в папку проекта `cd ./LibraryFund && npm i `

  * Запуск dev версии проекта  `npm start  // localhost:3000`

  * Собираем проект для продакшена  `npm run build `
